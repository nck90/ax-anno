> **삭제/변환 규칙**:
> - 모든 공백 삭제
> - 어미 "-임" 삭제 (맹지임→맹지, 자연림임→자연림)
> - "공유자의 우선매수신고는 1회에 한하여 행사할 수 있음" → "공유자우선매수신고1회제한"
> - "공부상 지목이 임야이나" → 삭제 (현황 정보만 남김)
> - 마침표 `.` 는 문장 구분으로 보존, 쉼표 `,` 보존

### 5.5 비고란이 비워지는 경우

2. 건물 구조 키워드 삭제 (경량철골구조, 슬래브지붕 등)
3. "제2종근린생활시설" → "근린시설"
4. 제시외 건물 개별 내역 → 합산 축약 (`첫항목명등합산면적㎡`)
5. 모든 공백 삭제 (비고, 소재지 부가정보)
6. 어미 삭제 (-임, -요망→-요)
7. 괄호 변환 `()` → `[]`
8. 분수 표기 변환 ("N분의 M" → "M/N")

### 데이터 재배치
1. 비고의 물건특성 → 소재지 `[...]`
2. 다중필지 사건의 전체주의사항 → 마지막 필지 `[물건번호N:...]`
3. 농지취득자격증명 문구: 비고 내용이 그것뿐이면 소재지로 이동, 아니면 비고에 축약 잔류
4. 동일 동/리 반복 필지 → "동소" 축약
